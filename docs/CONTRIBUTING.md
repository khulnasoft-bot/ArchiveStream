# Contributing to ArchiveStream

First off, thank you for considering contributing to ArchiveStream! It's people like you that make ArchiveStream such a great tool.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct.

## How Can I Contribute?

### Reporting Bugs

Bugs are tracked as GitHub issues. When creating a bug report, please include:
*   A clear and descriptive title.
*   The exact steps which can be reproduced.
*   Your OS and version.

### Suggesting Enhancements

Enhancement suggestions are also tracked as GitHub issues.

### Pull Requests

1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure the test suite passes.
4.  Make sure your code lints.
5.  Issue that pull request!

## Development Setup

### Rust

```bash
cargo build
cargo test
```

### Next.js

```bash
cd apps/web-ui
npm install
npm run dev
```
